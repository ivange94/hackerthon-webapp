# Hackerthon Webapp

Front end for the hackerthon app

## Setup

### Fork repo

### Clone your fork

  `git clone https://github.com/YourGitHubId/hackerthon-webapp`
    
### Install dependencies

  `cd hackethon-webapp`

  `npm install`

### Launch dev server

   `ng serve`
   
Your application will be running on http://localhost:4200
